# Shared Office Booking System

This is a web application for booking shared office spaces.
The project is currently under development while learning through tutorials.


## Tech Stack

- Database: SQL Server  
- Backend: Spring Boot  
- Frontend: Vue.js  
- Communication: Axios
- Authentication & Encryption: JWT
- Data Transfer: DTO
- Architecture: RESTful API  
- Testing Tool: Postman  

## Features Completed

- User login (JWT-based authentication)
- CRUD operations for Room management

## Features in Progress / To-Do

- User roles and access control
- Booking time slot functionality
