Shared Office Booking System

A web application for booking shared office spaces.
This project is currently under development as part of my learning process through tutorials.

Tech Stack
- Database: SQL Server
- Backend: Spring Boot
- Frontend: Vue.js
- Communication: Axios
- Authentication & Encryption: JWT
- Data Transfer: DTO
- Architecture: RESTful API
- Testing Tool: Postman

Features Completed
- User login with JWT-based authentication
- CRUD operations for room management

Features In Progress / To-Do
- User roles and access control
- Booking time slot functionality
